# Divyam CLI

Divyam command line tool to manage and configure Divyam and verify Divyam
installations.

## Documentation

Please refer to the [wiki](https://github.com/Divyam-AI/divyam-cli/wiki) for
installation, user and developer docs.

### macOS installation

- [docs/macos-install.md](docs/macos-install.md) - Homebrew Cask and standalone installer commands, including the unsigned-binary trust step.

### Gemini model provider setup

- [docs/gemini-model-info.md](docs/gemini-model-info.md): `model-info create` examples for AI Studio, Vertex SA key, Vertex ADC, and impersonation (WIF)
- [Gemini model provider config](https://github.com/Divyam-AI/divyam_router_controller/blob/main/docs/gemini_model_provider_config.md): API fields and GCP `gcloud` commands
